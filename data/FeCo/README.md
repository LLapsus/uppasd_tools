# FeCo Binary alloy

An example of FeCo binary alloy. The UppASD input files originate from [UppASD test folder](https://github.com/UppASD/UppASD/tree/d8d666328afedcd588c1f610d958fa259d964a82/tests/FeCo).

## Monte Carlo Simulation

- **initial phase:** 5 stages with decreasing temperatures (500K, 400K, 300K, 200K, 100K) each with 2000 MC steps
- **main phase:** 5000 MC steps at temperature 500K
