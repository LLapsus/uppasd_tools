# BCC Iron

Simple Monte Carlo simulation of BCC iron. The input files originate from [UppASD test folder](https://github.com/UppASD/UppASD/tree/d8d666328afedcd588c1f610d958fa259d964a82/tests/bccFe).

## Monte Carlo Simulation

- **initial phase:** 3 stages with decreasing temperatures (1000K, 700K, 500K) each with 2000 MC steps
- **main phase:** 3000 MC steps at temperature 500K
